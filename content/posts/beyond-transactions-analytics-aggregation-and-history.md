# Beyond Transactions: Analytics Aggregation and History

The first shift from a Transaction System to an Analytics System is a change in perspective: from the microscopic "row" to the macroscopic "insight."

| | Transactions | Analytics |
| :--- | :--- | :--- |
| **Purpose** | Record business operations | Instant summarized and trend business insights |
| **Data** | Single-row reads and writes | Multi-level pre-aggregation data |
| **Longevity** | Current state (Purge old) | Historical data for trend analysis |

### From State to Story
In a Transactions System, old data is a liability - it slows down the index and increases storage costs. In an Analytics System, old data is the most valuable asset. Without the "history," you don't have a trend; you only have a snapshot.
